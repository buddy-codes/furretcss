furret.css

My personal website styling toolkit, modeled after the lovely Water.css by Kognise. Written to allow quick and beautiful styling for simple document-like websites. Grab a copy here.

Or more simply, paste the following into the <head> of your HTML: 

<link rel="stylesheet" href="https://css.furret.codes/css/library.min.css" />

In addition to a light and dark theme, furret.css can also be extended using CSS variables in the :root element (see below). 

:root {
  --color-strong: #000;
  --color-main: #434242;
  --color-dim: #737373;

  --accent-strong: #0676a2;
  --accent-main: #7ddaff;
  --accent-dim: #a8dff5;

  --highlight-color: rgba(125, 218, 255, 0.8);
  --selection-color: rgba(125, 218, 255, 0.2);

  --background-dark: #e3e3e3;
  --background-darker: #f3f3f3;
  --background-darkest: #fff;
}
